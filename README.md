# Solo-Bookkeeping

一款記帳程式，使用 Material Design 風格來做設計。主要頁面有以下四個

- 記帳頁面
- 欠賬頁面
- 統計頁面
- 帳號頁面

## 記帳

- 圓餅圖讓用戶清楚的知道當月各類別支出佔本
- 顯示每日金流細項

## 欠賬

- 顯示當前獲得和借出的金額
- 顯示獲得和借出對比，讓用戶大概了解比例
- 根據獲得和借出的比例顯示不同表情，讓用戶身歷其境
